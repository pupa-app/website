How “Open in Pupa” works

A .pupa bundle is inert, portable data — no code runs on import. Tapping Open in Pupa downloads the bundle; your device then offers to open it in Pupa, which shows a confirm preview so you can review the app’s agent prompts before installing.

  • iOS: tap → “Open in Pupa” from the download.
  • macOS: the file downloads — open it to launch Pupa.
